Esta biblioteca se destina a validar números de cpf.

## Como instalar:

```
$  npm install lab-cpf-validator
```

## Como utilizar:
*Formato de entrada da função: sting.*

Exemplo:

```
> const cpf = require('lab-cpf-validator');
> console.log(cpf('12345678909'));
> // returns true
```
